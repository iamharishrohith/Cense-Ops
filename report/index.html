Section 04

The 10 Innovations

Each one addresses a gap that no existing platform fills.

1

Virtual DB Architecture

Redis ↔ PostgreSQL Reconciler — inspired by React's Virtual DOM

Redis acts as the "Virtual DB" for sub-millisecond dashboard reads. PostgreSQL is the audit-grade "Real DB." A Reconciler diffs dirty keys every 5 seconds and batch-writes only the delta. Nobody applies the Virtual DOM pattern to database architecture — this is cross-domain innovation.

2

AI Agent Cost Governance

"The Invisible Worker Tax" — tracking what no one else tracks

Monitors per-agent API token consumption and inference costs. Calculates ROI per AI agent. Detects redundancy (Copilot + Tabnine + Cursor doing the same job). Shows the total "AI Tax" per department.

"Your organization spent ₹73.3L this month on 10 AI agents. 4 overlap in Code Generation. Consolidating to Copilot alone saves ₹1.07Cr/year."
3

Department DNA Fingerprinting

Each department has a unique waste pattern — treat them differently

Engineering wastes on zombie dev environments. HR has orphaned licenses from leavers. Marketing has MarTech overlap. Finance has redundant ERPs. Each department head sees their specific waste pattern, benchmarked against industry norms.

5

Shadow AI Radar

Detecting the 50% of unapproved AI tool usage nobody sees

Scans expense reports, browser extensions, SSO anomalies, and network traffic. Classifies risk from CRITICAL (DeepSeek, personal ChatGPT) to LOW (Loom, browser-only tools). Visualized as a department × category risk heatmap.

6

License Autopilot

Self-healing governance with JML (Joiner-Mover-Leaver) automation

User inactive 14 days → automated Slack DM. No response in 48 hours → auto-reclaim license → notify manager. New hire → role-based provisioning. Role change → deprovision old, provision new. Departure → instant revocation of all access.

7

India-First Compliance Engine

DPDP Act + GFR + TNeGA — no western platform offers this

Automated DPDP consent management. PII masking in logs. 72-hour breach notification templates. GFR procurement workflows. TNeGA state-level integration. Event-sourced immutable audit trail.

Zero-Knowledge Privacy Discovery

The "Anti-Spyware" Agent — proves compliance by architecture, not policy

Browser extension checks URLs against a local SaaS signature database on the device. Matches send a signal. Non-matches are instantly destroyed at the edge — they never leave the device. PII never hits the server. DPDP data minimization taken to its mathematical extreme.

10

Carbon-Ops — Sustainability Score

SaaS waste isn't just financial — it's a carbon leak

Translates license waste into environmental impact. Reclaiming 1,000 unused licenses = 2,340 kg CO₂ reduced, 106 trees equivalent, 9,600 kWh saved. Aligns with India's Green IT Roadmap 2026-2030 and ESG mandates.

Technical Architecture

Bun.js runtime, Elysia.js backend, Next.js 15 frontend, Redis ↔ PostgreSQL Virtual DB engine.

Discovery Layer

SSO/IdP, Expense Reports, SaaS APIs, Browser Extensions, Network Analysis → Plugin Architecture (fetchUsage, fetchLicenses, reclaimSeat)

Virtual DB Engine

Redis (Hot): Live KPIs, usage snapshots, shadow detections, pub/sub → Reconciler (5s diff + batch upsert) → PostgreSQL (Cold): History, contracts, audit log, compliance

Intelligence Engine

Anomaly Detection, Renewal Risk Prediction, Consolidation Scorer, Savings Projector, AI Agent ROI Calculator, GeM Benchmarker, Carbon Calculator

Action Layer

Savings Simulator, License Autopilot, Shadow AI Radar, Negotiate Mode, GeM Benchmarking, Carbon-Ops, Compliance Engine

Presentation Layer

Executive Dashboard, SaaS Genome Map, Department DNA, Simulator UI, AI Monitor, Shadow Radar — all real-time via WebSocket ← Redis Pub/Sub
